# Swiggy Sales Analytics Dashboard (Excel)

## 📊 Project Overview
This project is an interactive Excel dashboard analyzing Swiggy sales data.

## 🔍 Features
- Total Sales, Orders, Ratings analysis
- Daily, Weekly, Monthly trends
- State-wise and city-wise performance
- Veg vs Non-Veg category insights

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Dashboard Design

## 📈 Insights
- Metro cities contribute highest sales
- Peak sales observed on weekends
- Non-veg category dominates sales
